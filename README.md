# Interactive European Energy Grid Connection Map

This website is ready for GitHub Pages.

## Files

- `index.html` = the website
- `data/countries.json` = editable country information
- `assets/` = optional folder for images or additional files

## How to edit country content

Open:

```text
data/countries.json
```

Edit the text for each country and save the file.

Then upload/commit the changed file to GitHub. GitHub Pages will update the website automatically after deployment.

## Important

Keep the folder structure unchanged:

```text
index.html
data/countries.json
assets/
```
