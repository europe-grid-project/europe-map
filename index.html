Hosting through GitHub Pages

This website is hosted using GitHub Pages, a service provided by GitHub. When the website is accessed, technical connection data may be processed by GitHub. This may include the visitor’s IP address, browser information, date and time of access, requested files and other technical log information required to provide and secure the service.

Further information is available in the GitHub General Privacy Statement .

External map services and technical resources

The website loads technical resources and map content from external providers. When these resources are loaded, the visitor’s browser connects directly to the respective provider. The provider may receive technical information, including the visitor’s IP address, browser information, requested resource and time of access.

  • Leaflet files are loaded through the unpkg content delivery network.
  • The map background is loaded from CARTO basemap servers.
  • Geographic country data is loaded from a GitHub-hosted GeoJSON file.
  • The basemap contains data based on OpenStreetMap contributors.
